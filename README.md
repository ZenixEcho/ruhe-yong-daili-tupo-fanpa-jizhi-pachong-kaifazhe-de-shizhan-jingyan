## 🚧 什么是反爬机制？

反爬机制（Anti-Scraping Mechanism）是网站为了防止数据被爬取而设置的防御系统，常见方法包括：

- 限制同IP访问频率
- 检测User-Agent / Cookie是否缺失
- 判断请求是否为自动化行为
- 加入验证码或JavaScript挑战
